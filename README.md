# Slider

jQuery slider plugin with four direction support and more

## Getting Started
[Demo page](file:///Applications/MAMP/htdocs/portal/qianfu/html/slider/demo/index.html)

In your web page:

```html
<script src="jquery.js"></script>
<script src="dist/aaslider.js"></script>
<script>
jQuery(function($) {
    $('.js-slider').slider({
          width: 979,
          height: 401,
          direction: 'right',
          duration: 1000,
          stopDuration: 1000,
      });
});
</script>
```
